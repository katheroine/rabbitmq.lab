# RabbitMQ.lab

Laboratory of RabbitMQ.

## Preparing

**composer.json**

```composer
{
    "require": {
        "php-amqplib/php-amqplib": "^3.5"
    }
}
```

**Dependencies installing**

`$ composer install`
